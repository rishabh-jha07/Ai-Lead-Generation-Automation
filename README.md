# AI Workflow Automation using Make.com

## Overview

This project is an AI-powered workflow automation built using **Make.com**. It automates repetitive tasks by integrating AI with multiple applications, improving efficiency and reducing manual effort.

## Features

* Automated end-to-end workflow
* AI-powered data processing
* Integration with multiple applications
* Reduces manual work and saves time
* Easy to customize and extend

## Technologies Used

* Make.com
* OpenAI API
* HTTP Module

## How to Use

1. Download the blueprint file.
2. Log in to your Make.com account.
3. Import the blueprint into a new scenario.
4. Configure the required API keys and connections.
5. Run the scenario and test the automation.

## Project Goal

The objective of this project is to demonstrate how AI can automate business workflows, reduce repetitive tasks, and improve productivity using a no-code platform.
